# Angular 1.x map

### [Demo][1]

## Requirements

1. Bower
2. NodeJS
3. Compass and sass gems

## Install 

1. Clone this repository
    ```
    git clone git@github.com:gatodeveloper/angular-app.git
    ```
2. Install dependencies
    ```
    npm i && bower i
    ```
3. Run project
    ```
    gulp
    ```
4. View on your browser
	```
	http://localhost:4001
	```
